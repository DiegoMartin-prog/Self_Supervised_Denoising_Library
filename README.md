# Self Supervised Denoising Library
---

This is a small project that I'll work on slowly while on my free time.  
The goal is to create a library where I collect and test some of the most famous image denoising self-supervised approaches in literature.  

This project is also intended to be used as a learning tool for my specific growth as a computer scientist, hence there might be some changes I'll try to add to such models.
In particular, I plan to start by reproducing the following papers: 
- Noise2Noise -> From my understanding this is the beginning of the entire field. I would like to test its performance with the plain old models, but also when the technique is applied to newer models such as ConvNext and ViTs.
- Noise2Voide -> This is the first and initial paper behind the whole field of self-supervised image denoising. It is the staple of the field, hence I might reproduce it and test it also with old and new models.
